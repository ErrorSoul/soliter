# Card Style System — Plan

## Идея
Карты можно переключать между разными визуальными стилями без изменения логики игры.
Стиль — это набор спрайтов для одного и того же набора карт.

## Сохранённые варианты (мокапы)

| Вариант | Файл | Описание | Статус |
|---------|------|----------|--------|
| B1 — Chinese Clean | `card_mockup_B1_chinese_clean.html` | Китайские масти (万/索/筒), чистый CSS стиль | ✅ Мокап готов |
| B2 — Hand Drawn | `card_mockup_B2_handdrawn.html` | Те же масти, hand-drawn/ink стиль с SVG roughness | ✅ Мокап готов |

Мокапы лежат в `~/works/selectel/`

## Структура карт (общая для всех стилей)

Каждый стиль состоит из одинакового набора файлов:
```
cards/
  red_cards/   2_red.png ... 10_red.png
  green_cards/ 2_green.png ... 10_green.png
  blue_cards/  2_blue.png ... 10_blue.png
  d_red.png, d_green.png, d_blue.png   (драконы)
  f_flower.png                          (цветок)
  back.png                              (рубашка)
```
Итого: 27 + 3 + 1 + 1 = **32 файла** на стиль.

## Реализация переключения стилей

В Defold карты загружаются через atlas. Для переключения стилей:
1. Держать несколько atlas-файлов (по одному на стиль)
2. При старте игры/в меню — выбрать активный atlas
3. Передать через `shared state` или `game.project` custom property

(Детали реализации — отдельная задача, сначала нужен хотя бы один готовый стиль)
